# Application Features

The Mental Health Companion is designed with a feature-rich environment catering to two distinct user roles: the Standard User (Patient) and the Medical Professional (Doctor).

## Core Features (Available to All Users)

### Secure Authentication
- Encrypted email and password registration.
- Persistent session management.
- Secure, token-based data access ensuring user privacy.

## Features: Standard User (Patient)

### 1. AI Therapy Sessions
- 24/7 access to the Gemini-powered CBT chatbot.
- Conversation history retention for context-aware responses.
- Automatic crisis detection and emergency routing.

### 2. Clinical Assessments
- Digital PHQ-9 (Depression) and GAD-7 (Anxiety) questionnaires.
- Real-time scoring and severity categorization.
- Graphical tracking of historical scores.

### 3. Medical Directory & Booking
- Searchable directory of verified mental health professionals.
- Interactive calendar and time-slot selection for appointment booking.
- Dashboard for viewing upcoming and past scheduled sessions.

### 4. Community Engagement
- Anonymous viewing and posting in categorized mental health forums.
- Ability to engage with peers through threaded comments and reaction systems.

### 5. Emergency Resources
- One-tap dialing for national crisis and suicide prevention helplines.
- Access to an embedded library of therapeutic PDFs and video content.

## Features: Medical Professional (Doctor)

### 1. Professional Dashboard
- Dedicated home interface specifically designed for practitioner workflows.
- Analytics and overview of active patients.

### 2. Appointment Management
- View requested appointments from standard users.
- Approve, modify, or cancel upcoming clinical sessions.
- Review patient assessment histories (with user consent) prior to sessions.
